## Enable Dreaming (Required First!)

Dreaming is disabled by default. To enable in Control UI:

1. Add to `plugins.entries` in openclaw.json:
```json
"memory-core": {
  "enabled": true,
  "config": {
    "dreaming": {
      "enabled": true,
      "frequency": "0 * * *"
    }
  }
}
```
2. Restart gateway
3. Dreams tab in Control UI will become active
