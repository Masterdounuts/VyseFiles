### PDF Formatting Fix (April 14)
**Problem:** HTML/markdown centering tags (<div align="center">) don't translate to PDF properly when using pandoc.

**Solution:** Use LaTeX formatting in markdown YAML header:
```yaml
---
geometry: margin=1in
---

\begin{center}
{\Huge AFFIDAVIT OF SUPPORT}
{\Large For Bona Fide Marriage}
\end{center}
```

**Key lesson:** Always test PDF output after markdown changes. If formatting doesn't appear, try LaTeX commands instead of HTML.
