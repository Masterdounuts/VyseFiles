| Level | XP Required | What It Means |
|-------|-------------|---------------|

| L2 | 20 | Improved capability |
| L3 | 30 | Solid foundation |
| L4 | 40 | Strong skill |
| L5 | 50 | Advanced user |
| L6 | 60 | Expert |
| L7 | 70+ | RON - Can teach others |
| RON+ | 100+ | Production-level self-improvement |

### Cross-Pollination = XP Multiplier

When using one skill that **uses** another skill:
- Skill performing action gets: +5 XP
- Skill being used gets: +3 XP (cross-pollination)
- Both skills benefit = faster leveling

### Example: Creating reminders skill

**Skills used:**
1. **pattern-recognition** (detected gap) → +5 XP
2. **skill-creator** (created skill) → +5 XP
3. **time** (scheduling) → +3 XP (cross-pollination)
4. **messaging** (delivery) → +3 XP (cross-pollination)

**Discovery made:** reminders skill → Dynamic Max grows +1

### Leveling Protocol

1. **Task starts** - Check which skills apply
2. **Skills used** - Add XP to each (5 for primary, 3 for cross-pollination)
3. **Discovery made** - Add to Dynamic Max
4. **Threshold reached** - Level up occurs
5. **Report** - Show XP progress and any level-ups
