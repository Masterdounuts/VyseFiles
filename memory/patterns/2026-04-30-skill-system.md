## Full Skill System Understanding

**Documented:** 2026-04-30
**Category:** skill-system

### The Three Pillars

1. **RON Level (1-7 base)**
   - Traditional skill progression
   - Level 7 = RON (can teach others)
   - Each skill has sub-skills leveling independently

2. **Dynamic Max (8-15)**
   - Ceiling that EXPANDS with discoveries
   - Core engines: 10-11 (learning, memory, pattern-recognition)
   - Meta skills: 10 (skill-creator, subagent-creator)
   - Regular skills: 8-9

3. **Cross-Pollination**
   - When one skill levels, related skills may update
   - Pattern: skill → related skill → sub-skill
   - Example: pattern-recognition → memory → storage

### The Skill Chain

```
DISCOVERY (pattern-recognition)
    ↓
DOCUMENT (memory)  
    ↓
LEARN (learning)
    ↓
TEACH (skill-creator → other skills)
    ↓
IMPROVE (system, self-healing)
```

### 24 Skills × Sub-skills = Massive Growth Potential

Each skill has 3-6 sub-skills that can level independently, all contributing to the main skill level and Dynamic Max.

### Level-Up Triggers

| Action | Skills Affected |
|--------|-----------------|
| Analyze patterns | pattern-recognition, learning |
| Document discovery | memory, knowledge |
| Fix issues | system, shipwright, self-healing |
| Create something | skill-creator, subagent-creator |
| Communicate | crew-protocols, messaging |

## Pattern: Skills Level When Used

When analyzing the skill system, multiple skills are engaged:
- pattern-recognition: Detecting patterns in skills
- knowledge: Understanding definitions
- learning: Recording the understanding
- memory: Storing pattern logs
- system: Checking cron/jobs
- exec: Running commands
- crew-protocols: Communication patterns

This multi-skill engagement triggers cross-pollination and level-ups.
