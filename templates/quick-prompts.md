# Quick Prompt Library

Reusable prompts for common tasks.

---

## Stock Check
```
Check current stock prices for {STOCK}. Compare to last known price. Alert if moved >5%.
```

## Dream to Content
```
Take today's dream entry from /root/.openclaw/workspace/DREAMS.md and convert it into a social media post. Include: 1) Hook, 2) Main insight, 3) CTA. Keep under 280 chars.
```

## Memory Update
```
Add the following to /root/.openclaw/workspace/memory/YYYY-MM-DD.md:
{KEY_EVENTS}

Then push to GitHub.
```

## Document Review
```
Review /path/to/document.pdf and identify: 1) Visual issues, 2) Content improvements, 3) Format changes needed.
```

## Daily Summary
```
Create a summary of today:
- Stocks: {PRICES}
- Decisions: {DECISIONS}
- Fixes: {FIXES}
- Next: {NEXT}
```
