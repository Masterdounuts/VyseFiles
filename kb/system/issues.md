# Issues

*Crew can flag problems here for Vyse to review*

## How to Use

1. **Flag an issue:** Add a new row below
2. **Priority:** 🔴 High (needs now) / 🟡 Medium (today) / 🟢 Low (when able)
3. **Status:** Open → In Progress → Resolved
4. **Vyse checks:** On every wake-up
