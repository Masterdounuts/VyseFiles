# Robinhood Android App Research

*Research date: April 14, 2026*

---

## What Robinhood Offers

### ✅ What We Can Do (Opportunities)
- **Stocks** - Buy/sell individual stocks (we have this)
- **ETFs** - Exchange-traded funds
- **Options** - Need to enable (David said not yet)
- **Cryptocurrency** - BTC, ETH, etc.
- **Limit Orders** - Set exact buy/sell price
- **Stop-Loss Orders** - Auto-sell if price drops
- **Market Orders** - Instant buy/sell at current price
- **Extended Hours Trading** - Pre-market & after-hours
- **Fractional Shares** - Buy partial shares (good for expensive stocks like TSLA!)
- **Gold** - Premium tier with more features

### ❌ Limitations
- **No advanced charting** - Basic charts only
- **No Level 2 data** - Can't see order book
- **No margin trading** (without Gold + approval)
- **Limited research** - Basic info, not pro-grade
- **No OTC stocks** - Over-the-counter limited
- **No mutual funds**
- **Settlement** - T+1 (can cause buying power issues if selling same day)

## Order Types Available

| Order Type | What It Does | Our Use |
|------------|--------------|---------|
| **Market** | Execute immediately at current price | ✅ Good for quick sells |
| **Limit** | Only execute at your price or better | ✅ Set sell targets |
| **Stop-Loss** | Auto-sell when price drops to level | ⚠️ Need to enable |
| **Stop-Limit** | Stop price triggers limit order | ⚠️ More control |
| **Trailing Stop** | Follows price, sells on drop | For volatile stocks |
