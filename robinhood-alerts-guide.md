## 📱 How to Set Price Alerts in Robinhood

### Method 1: From Search
1. Tap **Search** 🔍 → Type stock (e.g., "GGB")
2. Tap the **🔔 Bell icon** on stock page
3. Select **"Add alert"**
4. Choose condition → Enter price → Tap **Save**

### Method 2: From Portfolio
1. Tap **Portfolio** → Select stock
2. Tap **🔔 Bell icon** → "Add alert"

## 🔔 Enable Push Notifications

**Don't miss alerts!**
1. Go to **Account** (person icon) → **Notifications**
2. Enable **"Price alerts"** → Set to **"Push"**
3. Also enable **"Order updates"** for trade fills
