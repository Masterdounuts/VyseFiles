### Lessons Learned
1. Sandbox mode requires Docker - check before enabling
2. Pairing codes expire after 1 hour
3. Cron job delivery config can be fixed directly in jobs.json
4. CLI uses --url flag, not "link" command
5. Remote ws:// connections blocked by default (security) - need SSH tunnel or wss://
6. Git submodules embedded in repo cause issues - avoid nested repos
