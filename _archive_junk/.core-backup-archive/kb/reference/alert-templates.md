# Alert Templates

Quick message templates for common scenarios.

---

## Stock Big Move Alert

```
📈 {STOCK} Alert: {DIRECTION} {PERCENT}% today
Current: ${PRICE}
Consider: {ACTION}
```

## Stock Target Hit

```
🎯 {STOCK} Target Hit!
Current: ${PRICE}
Target: ${TARGET}
Action needed: Buy more / Sell / Hold?
```

## Cron Job Failure

```
⚠️ Cron Job Failed: {JOB_NAME}
Error: {ERROR}
Status: {status}
```

## Daily Summary

```
📋 Daily Summary - {DATE}
- Stocks: {PRICES}
- Decisions: {DECISIONS}
- Issues Fixed: {FIXES}
- Next: {NEXT}
```
