# Learning System

How Vyse learns and improves over time.

## Components

### Memory
- Short-term recall (~2500 entries)
- Long-term promotion (threshold: 3 recalls, 3 unique queries)
- Dreaming: Nightly at 3am UTC

### Self-Improvement
- Periodic self-audits
- Error tracking and correction
- Protocol refinement

## Dreams & Memory Palace

- **Dreaming**: Enabled, runs daily at 3am UTC
- **Memory Palace**: Wiki-backed knowledge vault
- **Short-term recall**: Tracks concepts across sessions

## Promotion Criteria

To be promoted from short-term to long-term memory:
- Minimum 3 recallCount
- Minimum 3 unique queries
- Score ≥ 0.75

## Self-Improvement Loop

1. Identify failure or inefficiency
2. Document root cause
3. Implement fix or new approach
4. Test and verify
5. Document outcome
